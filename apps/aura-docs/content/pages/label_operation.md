# Label / Operation

`Label` 是 Icon + Text 的基础组合控件，`Operation` 是左侧说明 + 右侧操作区域的两端对齐布局，适合设置项、控制面板和工具栏行。`Operation` 可选说明文本、状态标签、状态颜色、禁用态和无边框紧凑布局。
